## ğŸ Getting Started

To get the project running, follow these simple steps in your terminal:

1.  **Clone the repo**:
    ```bash
    git clone [https://github.com/pandeymehak217-lab/SpeedAI-Analyser.git](https://github.com/pandeymehak217-lab/SpeedAI-Analyser.git)
    cd SpeedAI-Analyser
    ```
2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run Analyzer on a Video (CLI Mode):**
    ```bash
    python main.py --video path_to_video.mp4
    ```
